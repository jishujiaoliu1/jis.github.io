What is code ?
==


The machines you use, such as computers and mobile phones, cannot understand what we say. Code is the language that machines can understand. All electronic products such as computers and mobile phones communicate in this language . If you want to use these electronic products to run/do things and make your own software/APP, you must communicate with them using code.Ordinary people cannot understand this language. Only programmers who have learned this language can communicate with computers.After programmers used this language to communicate with computers, they created various software for ordinary people who could not write code, so that ordinary people could also use electronic products.The other answers are too profound, I think mine should be easy to understand.
